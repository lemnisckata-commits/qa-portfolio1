## 📁 Что есть в репозитории

### 🔹 saucedemo/
Тестирование интернет-магазина [Swag Labs](https://www.saucedemo.com)
- `test-cases/` — тест-кейсы на авторизацию и корзину (15+ кейсов)
- `checklists/` — smoke-чеклист (20 пунктов)
- `bug-reports/` — 4 баг-репорта с реальными дефектами

### 🔹 demoblaze/
Тестирование магазина электроники [Demoblaze](https://www.demoblaze.com)
- `test-cases/` — тест-кейсы на авторизацию
- `checklists/` — smoke-чеклист (25 пунктов)
- `bug-reports/` — 4 баг-репорта включая API-баги

### 🔹 autotests/
Автоматизированные тесты на Python + Playwright
- 4 автотеста на авторизацию Swag Labs
- Запуск: `python -m pytest -v`

### 🔹 postman/
Коллекция API-тестов для Demoblaze
- Найденные баги: 500 на пустых полях, 200 вместо 401 при неверном пароле
